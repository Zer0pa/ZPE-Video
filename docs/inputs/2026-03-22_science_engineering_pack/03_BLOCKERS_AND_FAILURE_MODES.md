- Aggressive-side collapse: suppression approaches the target, but retained utility falls below the gate.
- Control-plane saturation: newer scorers rearrange decisions over the same packet without changing the underlying information content enough.

- Split authority surface: the live `.gpd` and `zpe_video_lab` evidence is in the outer workspace, while GitHub tracks the inner repo.
- Schema drift across phase outputs: similarly named summaries do not always share the same key layout or verdict encoding.
- Hidden regression risk: if compatibility files are removed faster than new measurement code is integrated, the inner repo becomes half-refactored and stops being a reliable handoff surface.

## What Is Not A Real Blocker

- More documentation that does not change the measured authority surface
- Treating a bounded frontier lift as if it just needs better narration
